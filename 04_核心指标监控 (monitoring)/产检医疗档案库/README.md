# 产检医疗档案库（索引）

## 目标
所有检查单据统一电子化归档，做到“随时可查、异地可共享、紧急可调取”。

## 命名规范（必须统一）
`YYYY-MM-DD_医院_项目_结论简述.jpg/pdf`

示例：
- `2026-04-13_周口市中心医院_hCG-孕酮_待复查.pdf`
- `2026-05-06_周口市中心医院_B超_宫内见胎心.jpg`

## 关键要求
- 每次产检后 24 小时内上传并登记。
- 纸质原件放入“家庭证件夹”，电子版保存到本目录。
- 男方和妻子各保留一份可访问副本，防止单点丢失。
