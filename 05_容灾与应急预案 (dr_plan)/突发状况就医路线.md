# 突发状况就医路线（见红/腹痛等）

## 适用场景
- 孕早期出现见红、持续腹痛、发热、晕厥等需要立即就医的情况。

## 路线优先级
### 路线A：就近急诊（最快）
- **目标**：沈丘县人民医院急诊
- **使用时机**：夜间、恶劣天气、症状突然加重
- **出发方式**：网约车优先，无法出行立即拨打 120

### 路线B：产科专项（条件允许）
- **目标**：周口市中心医院产科急诊/妇产科
- **使用时机**：症状相对稳定，但需要更高等级产科处置
- **出发方式**：助理或阿姨陪同前往，丈夫远程跟进

## 出发前 3 分钟动作清单
- [ ] 带上身份证、医保卡、既往检查单（手机电子版也可）
- [ ] 手机充电 >= 30%，移动电源随身
- [ ] 通知丈夫 + 助理/阿姨 + 预定医院
- [ ] 穿防滑鞋，避免单独行动

## 途中处理
- 尽量侧卧或半躺，避免紧张和剧烈走动。
- 不自行服药，不喝来路不明“偏方”。
- 将症状开始时间、出血量、疼痛强度简单记录给医生。

## 到院后沟通模板
“孕早期，今天出现（见红/腹痛/发热），开始于 XX:XX，持续约 X 小时，既往化验有 hCG/孕酮结果，担心先兆流产风险，请优先评估。”
