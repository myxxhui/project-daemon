## 二、异地养胎生活指南

### 2.1 明确结论：现阶段不适合飞菲律宾

> 孕 4–12 周是胚胎器官分化最关键时期，也是自然流产风险最高阶段（约 10–15%）。长途飞行（沈丘→郑州→马尼拉，约 8–10h）带来的辐射暴露、气压变化、久坐血栓风险，以及异地就医不便，综合评估**弊大于利**。
>
> **结论：孕 12 周前，妻子留在沈丘静养，男主按原计划在菲律宾工作，通过远程陪伴 SOP 维持连接。**

### 2.2 居家静养核心原则

- [ ] **作息**：保证 8h 夜间睡眠 + 1h 午休；22:30 前入睡
- [ ] **活动强度**：以"不累"为标准——散步 ≤ 30 分钟/次，禁止搬重物（> 5kg）
- [ ] **禁止事项**：
  - [ ] 禁止清理猫砂（弓形虫风险，100% 交给阿姨）
  - [ ] 禁止抱猫狗或让宠物舔脸/手（TORCH 防范期）
  - [ ] 禁止长时间站立拍摄（> 30 分钟连续站立）
  - [ ] 禁止剧烈运动、登高、搬运道具
  - [ ] 禁止泡温泉/热水浴（水温 > 38°C 可能影响胚胎）
  - [ ] 禁止接触化学染发/烫发产品
- [ ] **情绪管理**：孕早期激素波动大，情绪低落/焦虑属正常，但持续超过 2 周需告知产科医生
- [ ] **工作节奏**：自媒体工作以"脑力输出为主、体力输出为零"——选题、审稿可做；拍摄、搬运交给助理

### 2.3 就医升级红线（出现以下情况立刻就医，不要等）

- [ ] 明显见红 / 持续出血（超过护垫用量）
- [ ] 持续或加重下腹痛、坠胀感
- [ ] 发热 ≥ 38°C、寒战
- [ ] 剧烈孕吐导致无法进食超过 24h / 尿量明显减少
- [ ] 头晕、晕厥、心悸

### 3.2 孕早期饮食建议（清淡为主，少量多餐）

**推荐食物：**
- 主食：小米粥、白粥、全麦面包（易消化）
- 蛋白质：鸡蛋（全熟）、豆腐、瘦肉（全熟）、鱼（淡水鱼蒸熟）
- 蔬菜：绿叶菜（焯熟）、胡萝卜、南瓜
- 水果：苹果、梨、香蕉（孕吐严重时少量多次）
- 水分：每日 1500–2000mL，白开水为主

### 3.3 绝对忌口清单（孕期全程）

| 类别 | 禁止食物 | 原因 |
|---|---|---|
| 生食 | 生鱼片、刺身、溏心蛋、生肉 | 沙门氏菌、李斯特菌、弓形虫 |
| 酒精 | 所有含酒精饮品（包括米酒、料酒菜） | 致畸，无安全剂量 |
| 高汞鱼类 | 金枪鱼（大型）、旗鱼、鲨鱼 | 汞影响胎儿神经发育 |
| 未经巴氏消毒 | 生牛奶、自制奶酪、软奶酪 | 李斯特菌风险 |
| 高糖高脂 | 奶茶、蛋糕、油炸食品 | 孕吐加重、血糖波动 |
| 寒凉食物 | 冰饮、冰激凌、螃蟹、薏米 | 中医孕期禁忌，谨慎为主 |
| 刺激性食物 | 辣椒、浓咖啡（> 200mg 咖啡因/天）、浓茶 | 刺激子宫收缩 |
| 补品 | 人参、桂圆、鹿茸等热性补品 | 可能引起宫缩 |
| 药物 | 任何非医嘱药物（含中药、感冒药） | 致畸风险，必须问产科医生 |

## 四、我在菲律宾的远程陪伴 SOP

> **核心理念**：距离无法弥补，但"在场感"可以设计。用结构化的小仪式代替随机联系，让妻子感受到稳定的陪伴。

### 4.1 每日固定节奏

| 时间（北京时间） | 事项 | 形式 | 说明 |
|---|---|---|---|
| 08:00 | 早安打卡 | 微信语音/文字 | 问昨晚睡眠、今早孕吐情况 |
| 12:00–13:00 | 午间外卖投喂 | 美团/饿了么下单 | 提前问好想吃什么，备注"孕妇清淡少辣" |
| 20:00–21:00 | 晚间视频通话 | 微信/FaceTime | 固定 30 分钟，聊今天状态、明天计划 |
| 睡前 | 情绪打卡 | 微信文字 | 妻子用 1–5 分给今天情绪打分，男主回应 |

### 4.2 每日情绪打卡模板

妻子每晚睡前发一条：
```
今日情绪：⭐⭐⭐（1–5星）
孕吐程度：轻/中/重
今天吃了：________
身体不适：有/无（描述：________）
今天开心的事：________
明天想要：________
```
男主收到后必须回复，哪怕只是"收到，辛苦了，爱你"。

### 4.3 外卖投喂指南

- [ ] **美团/饿了么账号共享**或男主直接在 App 下单（开通异地下单）
- [ ] 每周一提前问本周想吃的菜系，列入"本周菜单"
- [ ] 优先选择：粥类、面条、蒸菜、炖汤（避免重油重辣）
- [ ] 备注栏固定写：**"孕妇用餐，请少盐少油不辣，食材全熟"**
- [ ] 每周至少 2 次"惊喜投喂"（不提前说，直接下单）

### 4.4 远程医疗协同

- [ ] 每次产检前：男主提前整理问题清单发给妻子带去问医生
- [ ] 每次产检后：妻子拍照上传报告，男主当天回看并记录到本文档
- [ ] 重要结果（hCG 翻倍、孕酮、B 超）：视频通话一起看报告，不要让妻子独自面对
- [ ] 紧急情况联系链：妻子 → 男主（电话）→ 员工小姑娘陪同就医 → 家政阿姨协助

### 4.5 情感支持备忘

- 孕早期妻子可能情绪波动大、敏感易哭，**不要讲道理，先共情**
- 每周至少说一次"你辛苦了，你做得很好"
- 避免在她孕吐/疲惫时讨论"大事"（财务、工作压力等）
- 可以定期寄小礼物（零食、花、小玩具）——快递比语言更有重量感
