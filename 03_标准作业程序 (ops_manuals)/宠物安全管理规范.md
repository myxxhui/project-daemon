# 宠物安全管理规范（孕期最高安全铁律）

> **适用范围**：家中 5 只猫 + 1 只杂交小型金毛犬；孕妇孕早期（0–12 周）起全程执行，直至产后医生解除限制。
>
> ⚠️ **本文档为家庭最高安全铁律，任何人不得以"麻烦""没必要"为由降低执行标准。违反者须向男主（丈夫）报告。**

---

## 一、医疗排雷：弓形虫（TORCH）检查计划

### 1.1 为什么这是第一优先级

弓形虫（*Toxoplasma gondii*）的主要传播宿主是猫科动物。猫粪中的弓形虫卵囊在环境中可存活数月。孕妇初次感染弓形虫可导致**流产、死胎、胎儿脑积水、视网膜炎**等严重后果。家中有 5 只猫，这是本家庭孕期最大的感染风险点，必须优先处置。

### 1.2 孕妇检查计划

| 检查项目 | 时间 | 医院 | 结果 | 处置 |
|---|---|---|---|---|
| TORCH 全套（含弓形虫 IgG/IgM） | 本周内（越快越好） | 周口市中心医院妇产科 |  |  |
| 弓形虫 IgG 复查（如 IgM 阳性） | 按医嘱 |  |  |  |

**结果解读（供参考，以医生判断为准）：**
- IgG(+) IgM(-) → 既往感染，已有免疫，风险较低，但仍需执行隔离措施
- IgG(-) IgM(-) → 无免疫，**高风险**，必须严格执行全部隔离措施
- IgM(+) → 可能近期感染，立即就医，由产科医生评估处置方案

## 二、物理隔离：禁区设立与活动限制

### 2.1 主卧"无宠禁区"（最高优先级）

- [ ] **主卧室**：即日起设为无宠物区，门保持关闭
- [ ] **婴儿用品存放区**：宠物禁止进入
- [ ] **孕妇餐食区（餐桌/厨房操作台）**：宠物禁止上桌/台面，发现立刻驱离
- [ ] **执行方式**：在主卧门口安装简易挡板或关门，猫咪跳跃能力强，需确认门能关严

### 2.2 狗的活动限制

- [ ] 狗的主要活动区域：**二楼阳台**（已有排泄习惯，保持现状）
- [ ] 狗下楼活动时：必须有阿姨在场监督，孕妇不单独与狗接触
- [ ] 禁止狗进入：主卧、孕妇休息区、厨房
- [ ] 禁止狗舔孕妇的手、脸、食物

### 2.3 猫的活动管理

- [ ] 猫可在公共区域活动，但主卧关闭
- [ ] 猫咪上床/上沙发后，孕妇须洗手再接触食物
- [ ] 孕妇可以抚摸猫，但**抚摸后必须洗手**，禁止让猫舔脸/嘴
- [ ] 猫砂盆放置位置：远离厨房和孕妇休息区（建议放卫生间或阳台）

## 三、日常运维交接 SOP

> **铁律：孕妇绝对禁止接触猫砂、宠物排泄物及相关清洁用品。此项工作 100% 由家政阿姨承担。**

### 3.1 猫砂清理 SOP（每日 2 次：早 8:00 / 晚 20:00）

**所需物品：**
- 一次性手套（每次用新的，用完即弃）
- 一次性口罩
- 猫砂铲
- 密封垃圾袋
- 洗手液（抗菌型）

**操作步骤：**
1. 戴好口罩和一次性手套
2. 用猫砂铲将结块/粪便捞出，放入密封垃圾袋
3. 系紧垃圾袋，放入室外垃圾桶（不留在室内）
4. 猫砂量不足时及时补充
5. 脱手套（由内向外翻转），丢入垃圾袋一并处理
6. **用洗手液彻底洗手至少 20 秒**，冲洗干净
7. 开窗通风至少 10 分钟

**每周深度清洁（周日）：**
- 将猫砂全部倒出，猫砂盆用热水 + 宠物消毒液刷洗
- 晾干后重新铺新猫砂
- 操作后同样彻底洗手

### 3.2 狗的排泄清理 SOP（每日至少 1 次，建议早晚各 1 次）

1. 戴一次性手套
2. 用纸巾/宠物专用清洁袋收集粪便，密封处理
3. 用清水冲洗阳台地面，必要时用宠物消毒液
4. 脱手套，彻底洗手

### 3.3 喂食与饮水管理

| 宠物 | 喂食频次 | 食物类型 | 禁止事项 |
|---|---|---|---|
| 5 只猫 | 每日 2 次（早/晚）+ 自动喂食机补充 | 猫粮（干粮为主，湿粮为辅） | **禁止喂生肉、生鱼** |
| 1 只狗 | 每日 2 次（早/晚） | 狗粮 | **禁止喂生肉、骨头、葡萄、洋葱** |

- [ ] 猫粮/狗粮库存不足时提前 3 天告知，由男主网购或阿姨采购
- [ ] 饮水碗每日换新鲜水，每周清洗碗具

### 3.4 宠物用品清洁周期

| 用品 | 清洁频次 | 方法 |
|---|---|---|
| 猫碗/狗碗 | 每日清洗 | 热水 + 洗洁精，冲净晾干 |
| 猫砂盆 | 每周深度清洁 | 见 3.1 |
| 猫咪窝/垫子 | 每 2 周清洗 | 机洗，60°C 以上水温 |
| 狗窝/垫子 | 每 2 周清洗 | 同上 |
| 猫抓板/玩具 | 每月检查更换 | 破损及时更换 |

## 四、饮食管理：严禁生肉

- [ ] **禁止给猫狗喂食任何生肉、生鱼、生内脏**（弓形虫、沙门氏菌的主要传播途径）
- [ ] 猫狗食物与人类食物严格分开存放（冰箱分层，宠物食物放最底层）
- [ ] 处理宠物食物后必须洗手，再接触孕妇餐食
- [ ] 禁止让宠物接触孕妇的餐具、食物

## 五、异常情况处理

### 5.1 宠物健康异常

| 情况 | 处置 | 联系人 |
|---|---|---|
| 猫/狗腹泻、呕吐超过 24h | 联系兽医，隔离病宠 | 兽医：________ |
| 宠物出现神经症状（抽搐、失去方向感） | 立即就医 | 同上 |
| 宠物咬伤/抓伤孕妇 | 立即用流水冲洗伤口 15 分钟，就医评估是否需要打疫苗 | 男主 + 产科医生 |
| 宠物逃跑/走失 | 通知男主，发布寻宠信息，孕妇不参与外出寻找 | 男主 |

### 5.2 孕妇意外接触猫砂/排泄物

1. 立即用流水 + 洗手液彻底洗手
2. 告知男主
3. 下次产检时告知医生，评估是否需要复查弓形虫

## 六、执行监督机制

- **日报**：家政阿姨每晚向男主微信汇报当日猫砂清理情况（一句话即可）
- **周检**：男主每周视频检查宠物区域卫生状况
- **违规处理**：孕妇如因任何原因接触了猫砂，立即上报，不隐瞒
