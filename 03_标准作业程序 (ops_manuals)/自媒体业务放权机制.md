# 自媒体业务放权机制（孕期版）

> **账号**：抖音宠物博主（主理人：妻子）
> **助理**：19 岁女员工（全职）
> **生效时间**：孕期确认即刻生效，直至产后主理人恢复全职状态
>
> **核心原则：妻子只动脑，不动身。所有体力型工作 100% 下放给助理。账号可以慢，不能让妻子累。**

---

## 一、职责划分矩阵

### 1.1 妻子（主理人）——脑力输出，轻量参与

| 职责 | 具体内容 | 时间投入上限 |
|---|---|---|
| 选题方向 | 每周提供 2–3 个选题方向或关键词 | ≤ 30 分钟/周 |
| 大纲审核 | 审阅助理写的脚本大纲，给方向性意见 | ≤ 20 分钟/条 |
| 成片把关 | 看剪辑成片，给出"过/改"结论 | ≤ 15 分钟/条 |
| 出镜（可选） | 状态好时可少量出镜，不强求 | 单次 ≤ 20 分钟，每周 ≤ 1 次 |
| 账号大方向 | 月度内容方向、商务合作决策 | 按需 |

> ⚠️ 妻子当日孕吐严重或身体不适时，所有工作可顺延，助理不得催促。

### 1.2 助理——体力 + 执行，全权负责

| 职责 | 具体内容 | 备注 |
|---|---|---|
| **脚本撰写** | 根据主理人选题方向写完整脚本/大纲 | 提交妻子审核后执行 |
| **拍摄执行** | 布景、打光、抱猫狗配合拍摄、运镜 | 妻子不参与搬运道具 |
| **剪辑制作** | 粗剪 + 精剪 + 字幕 + 封面 | 提交妻子审核 |
| **发布操作** | 按审核通过的内容定时发布 | 发布前截图存档 |
| **评论区管理** | 日常回复、点赞互动 | 见评论区管理规则 |
| **快递/采购** | 收发快递、采购拍摄道具、猫粮狗粮 | 妻子不搬运任何物品 |
| **猫砂/狗粮** | 配合家政阿姨，拍摄相关内容时负责道具清洁 | 妻子绝对不接触猫砂 |
| **数据整理** | 每周整理播放量、涨粉、互动数据 | 周报形式发给妻子 |
| **商务对接初筛** | 接收合作邀约，整理后提交妻子决策 | 不得擅自承诺合作 |

## 二、协作流程

```
选题确认（妻子，每周一）
    ↓
脚本撰写（助理，2天内）
    ↓
脚本审核（妻子，当天或次日）
    ↓
拍摄执行（助理主导，妻子按需参与）
    ↓
剪辑制作（助理，2–3天）
    ↓
成片审核（妻子，当天给结论）
    ↓
发布（助理操作，按最佳时间段）
    ↓
数据追踪（助理，次日汇报）
```

**周节奏建议：**

| 时间 | 事项 |
|---|---|
| 周一 | 妻子确认本周 2–3 个选题 |
| 周二–周三 | 助理完成脚本，提交审核 |
| 周四 | 拍摄日（如妻子出镜，控制在上午精力好时） |
| 周五–周六 | 剪辑 + 审核 + 发布 |
| 周日 | 助理整理周报，妻子休息 |

## 三、孕期内容转型建议

### 3.1 内容方向调整

| 原有方向 | 孕期调整方向 | 理由 |
|---|---|---|
| 宠物日常/搞笑 | 保留，但减少需要大量体力的内容 | 核心受众不流失 |
| 宠物技能/训练 | 助理主导拍摄，妻子画外音/配文 | 妻子不参与体力操作 |
| **新增：怀孕与宠物陪伴** | 温情向：猫咪趴在孕肚上、宠物守护孕妈 | 差异化内容，情感共鸣强 |
| **新增：孕期日常 vlog** | 轻量记录：产检、孕吐、宠物陪伴 | 真实感强，涨粉潜力大 |

> 新方向建议：**"怀孕了，5只猫1只狗怎么办？"** 这个话题本身就是流量点，可以做成系列内容（弓形虫科普、宠物与孕妇相处、宠物感知孕妈等）。

### 3.2 评论区控评策略

**助理日常评论区管理规则：**

| 评论类型 | 处理方式 |
|---|---|
| 正常互动、提问 | 助理正常回复 |
| 夸奖、祝福 | 点赞 + 简短回复 |
| 关于孕期宠物的担忧（善意） | 回复科普内容（弓形虫已检测等） |
| 轻微负面（质疑内容质量等） | 忽略，不回复，不删除 |
| **极端负面（诅咒、人身攻击、恶意诋毁）** | **直接拉黑，不截图给妻子看** |
| 涉及隐私/地址/家庭信息的评论 | 立即删除，不回复 |
| 商务合作私信 | 整理后每周汇报给妻子，不擅自回复 |

> ⚠️ **核心原则：妻子不看评论区。** 助理负责过滤所有负面内容，保护妻子情绪。妻子如果想看，助理提前截图正面评论给她看。

## 四、紧急熔断机制

### 4.1 触发条件

以下任一情况出现，立即启动熔断：
- [ ] 妻子出现严重孕吐（无法进食超过 24h）
- [ ] 妻子出现先兆流产迹象（见红、腹痛、医生要求卧床）
- [ ] 妻子情绪严重崩溃，无法正常沟通
- [ ] 男主判断需要暂停（远程指令）

### 4.2 熔断执行方案

**级别一：减速（妻子轻度不适）**
- 本周发布频次从 X 条降至 1 条
- 助理使用已有存货素材剪辑发布
- 妻子不参与任何审核，助理自行判断发布

**级别二：停更（妻子中度不适，需卧床休息）**
- 账号暂停更新
- 助理在评论区置顶一条：「主理人身体调整中，暂停更新，感谢陪伴 🌿」
- 不透露孕期信息（除非妻子主动决定公开）
- 已签约商务合作：助理联系品牌方说明情况，协商延期

**级别三：代发存货（妻子住院/需要长期休息）**
- 助理整理历史未发布素材，剪辑成可发布内容
- 每周发布 1–2 条，维持账号活跃度
- 所有发布内容不涉及妻子近况
- 男主远程审核，微信确认后发布

### 4.3 熔断解除条件
- 妻子身体恢复，主动表示可以恢复工作
- 男主确认解除

## 五、权限与账号安全

| 权限 | 持有人 | 说明 |
|---|---|---|
| 账号密码 | 妻子（主理人） | 不共享给助理 |
| 发布权限 | 助理（通过授权设备） | 仅可发布，不可修改账号信息 |

| 商务合作决策 | 妻子 | 助理不得擅自承诺 |
| 账号注销/转让 | 妻子 | 任何情况下不授权助理 |

| 维度 | 评分标准 |
