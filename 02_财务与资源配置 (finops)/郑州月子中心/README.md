# 郑州月子中心（调研主表）

## 调研目标
- 对齐 3-5 家候选机构，按“医疗安全 > 护理质量 > 距离便利 > 价格”的顺序决策。
- 提前完成签约与入住衔接，避免孕晚期临时拍板。

## 选型维度（建议权重）
| 维度 | 权重 | 说明 |
|---|---:|---|
| 医疗协同能力 | 30% | 合作医院距离、夜间转诊效率、突发处理经验 |
| 护理团队质量 | 25% | 护士/护理师资质、夜班配置、一对几比例 |
| 卫生与感染管理 | 20% | 消杀制度、探视规则、母婴分区 |
| 服务与饮食 | 15% | 产后恢复、哺乳支持、营养餐质量 |
| 价格与条款 | 10% | 套餐透明度、退费规则、隐形收费 |
