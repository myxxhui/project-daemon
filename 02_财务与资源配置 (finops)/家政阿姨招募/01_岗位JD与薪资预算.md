# 岗位JD与薪资预算（正式版）

## 需求摘要
- 服务对象：孕早期妻子（胚胎受孕时长约 2 周以内，临床孕周约 4 周左右）
- 家庭环境：240 平复式，5 猫 1 狗，丈夫长期在马尼拉远程协调
- 核心目标：孕妇零接触猫砂与宠物排泄物，饮食全熟安全，家务稳定可交接

## 岗位基本信息
| 项目 | 内容 |
|---|---|
| 岗位名称 | 住家阿姨 / 全职白班家政（二选一） |
| 工作地点 | 河南周口沈丘县（详细地址面谈） |
| 工作时间 | 住家：全天；白班：08:00-20:00（每周休 1 天） |
| 试工期 | 7-14 天（按日结或周结） |
| 转正合同 | 建议先签 3 个月短约，再评估续签 |
| 到岗时效 | 越快越好，优先可 3 天内上岗者 |

## 核心职责（必须执行）
### 1) 日常家务
- [ ] 每日扫拖、台面消毒、厨房清洁、垃圾清运
- [ ] 每周至少 1 次深度清洁（卫生间/阳台/宠物区）
- [ ] 衣物洗晒收纳（按“孕妇贴身衣物单独清洗”执行）

### 2) 孕期饮食
- [ ] 一日三餐 + 1-2 次加餐，清淡、少油、全熟
- [ ] 能根据孕吐变化调整菜单（粥、蒸蛋、面汤、炖汤等）
- [ ] 严禁提供生食、半熟食、含酒精烹饪、重辣重油

### 3) 宠物安全职责（最高优先级）
- [ ] 每日 2 次猫砂清理（戴手套+口罩，操作后彻底洗手）
- [ ] 每日清理狗阳台排泄物，保持无异味
- [ ] 每周清洗猫狗食具与猫砂盆，宠物食材不得用生肉
- [ ] 孕妇禁止接触猫砂和宠物排泄物，阿姨必须全程接管

### 4) 异常情况上报
- [ ] 孕妇出现见红/持续腹痛/发热 >= 38C，立即通知丈夫和助理并陪同就医
- [ ] 无法到岗须提前至少 4 小时报备并触发替补方案

## 任职条件
### 必须满足
- [ ] 不怕猫狗，能长期执行宠物清洁
- [ ] 身体健康，无传染性疾病（可提供近 6 个月体检）
- [ ] 性格稳定、沟通顺畅、可接受远程管理

### 优先考虑
- [ ] 有孕妇照护经验或月嫂经历
- [ ] 会做孕期清淡餐
- [ ] 沈丘本地、到岗稳定

### 一票否决
- [ ] 拒绝清理猫砂/狗排泄物
- [ ] 明显不理解孕期禁忌且拒绝学习
- [ ] 背景核查存在诚信问题
