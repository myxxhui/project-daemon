# 试工与转正评估

## 试工周期
- 建议时长：7-14 天
- 评估人：妻子 + 丈夫（远程）+ 助理
- 目标：验证“稳定性 + 执行力 + 应急能力”，不是只看态度

## 每日打分表（1-5分）
| 日期 | 准时到岗 | 卫生质量 | 饮食质量 | 宠物SOP执行 | 沟通反馈 | 应急响应 | 总评 |
|---|---:|---:|---:|---:|---:|---:|---:|
|  |  |  |  |  |  |  |  |
|  |  |  |  |  |  |  |  |
|  |  |  |  |  |  |  |  |

> 建议计算：`日总评 = 6项均分`，连续 3 天低于 3 分需预警沟通。

## 每日检查清单（必须打勾）
- [ ] 猫砂早晚两次按SOP执行并记录
- [ ] 狗阳台排泄已清理
- [ ] 孕妇三餐和加餐落实，且无禁忌食材
- [ ] 厨房和卫生间当天已清洁
- [ ] 当日异常情况已汇报（无异常也报“平安”）

## 试工期红线事件（出现即停）
- [ ] 让孕妇接触猫砂/排泄物
- [ ] 隐瞒突发情况或延迟上报
- [ ] 无故旷工或频繁迟到
- [ ] 明显顶撞或情绪失控影响照护

## 转正规则（建议）
- [ ] 连续一周无迟到早退
- [ ] 猫砂与宠物卫生执行无漏项
- [ ] 能独立处理突发情况并及时上报
- [ ] 妻子主观满意度 >= 4/5
- [ ] 丈夫远程协同满意度 >= 4/5

## 最终决定
- [ ] 通过转正
- [ ] 延长试工
- [ ] 不录用

## 备注
转正后建议保留每周一次简版评估，连续跟踪 4 周，防止“试工期好、转正后松懈”。
