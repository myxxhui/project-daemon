# 采购统筹清单（阶段化）

## 使用方式
按孕周阶段采购，避免一次性囤货过多造成浪费或买错规格。

## 阶段总览
| 阶段 | 重点采购 | 预算建议 | 状态 |
|---|---|---:|---|
| 孕早期（当前） | 叶酸、基础营养品、防滑拖鞋、分装药盒 | 500-1500 元 | 进行中 |
| 孕中期 | 托腹带、舒适内衣、基础待产品预采 | 1000-2500 元 | 未开始 |
| 孕晚期 | 待产包、产后护理品、婴儿高频消耗品 | 3000-8000 元 | 未开始 |

## 当前必买清单（孕早期）
- [ ] 叶酸（0.4-0.8mg/天）
- [ ] 维生素D（按医嘱）
- [ ] 家用体温计、电子血压计
- [ ] 防滑居家拖鞋、防跌扶手（浴室）
- [ ] 宠物清洁耗材（手套、口罩、密封垃圾袋）

## 采购原则
- 非刚需不提前买“婴儿大件”。
- 优先买“复购型刚需”，次选“低损耗保值品”。
- 每次采购保留截图和订单号，月底做一次复盘。
